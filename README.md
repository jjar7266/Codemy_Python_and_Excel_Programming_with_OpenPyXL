# Codemy: Python and Excel Programming with OpenPyXL

This repository is a single‑folder learning archive created by **Jose "Joe" Ruiz** while following John Elder’s  
**Python and Excel Programming With OpenPyXL** course on Codemy.com.

The project documents the student’s progress through the course, including personally written code, teaching notes, annotated examples, and any Excel files used or created during the lessons. This is an educational reference, not an application project.

---

## 📘 Purpose

- Follow the Codemy.com course step by step  
- Record all code written by Jose "Joe" Ruiz during the learning process  
- Add personal teaching notes, explanations, and examples  
- Build a long‑term reference for Python and Excel automation  
- Maintain a simple, flat file structure for easy navigation  

---

## 📁 Project Structure (Single‑Folder Layout)

All files are stored in the root directory:

- `lesson_XX_description.py` — Code written by Jose "Joe" Ruiz for each lesson  
- `notes_XX.md` — Personal notes and explanations  
- `workbook_XX.xlsx` — Excel files used or created in lessons  
- `README.md` — Course overview and progress  
- `.venv/` — Virtual environment managed with `uv`  
- `pyproject.toml` — Project dependencies (OpenPyXL)  

This layout keeps all materials alphabetical, chronological, and easy to reference.

---

## 🧰 Environment

- Python environment created using `uv venv`  
- Dependencies installed with `uv add openpyxl`  
- All lesson files remain in the root directory
